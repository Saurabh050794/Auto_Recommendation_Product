## 📌 Overview

ShopAI is a lightweight, browser-based product recommendation system
that delivers smart, real-time suggestions from a dataset of 1000+
products --- without requiring any backend or server.

Built entirely using HTML, CSS, and JavaScript.

------------------------------------------------------------------------

## ✨ Key Features

-   Intelligent Search Engine\
-   Trending Products\
-   Category Filtering\
-   Dynamic Price Filtering\
-   Similarity-Based Recommendations\
-   Dark / Light Mode UI\
-   Responsive Design

------------------------------------------------------------------------

## 🧠 How It Works

User Query → Tokenization → Matching → Ranking → Display
